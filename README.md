# game-of-life

Demonstration of Conway's Game of Life

### Demonstration

https://ma38su.github.io/game-of-life/?rate=0.2

### Paramters

You can change some paramters using URL query.

``?rate=0.1&width=100&height=100``

- rate: float = 0.1
- width: uint = 100
- height: uint = 100
